# Elevate-Labs-Data-Analytics-Internship-Task4
Dashboard design using powerbi with superstore dataset
__Superstore Sales & Profit Analysis Dashboard__
------
__Objective__

To analyze sales data and create visualizations that provide meaningful business insights and support decision-making.

----

__Tools Used__
- Power BI / Tableau  
- Python (Pandas for data cleaning)

---

__Dataset__
- Superstore.csv (Sales dataset with orders, profit, discount, shipping, and regional data)

---

__Key KPIs__
- Total Sales  
- Total Profit  
- Total Orders  
- Profit Margin (%)  
- Average Discount  
- Average Delivery Days  

---

__Key Insights__
- High discounts reduce overall profitability  
- Furniture category generates losses despite high sales  
- Technology category is the most profitable  
- Some regions show consistent low performance  
- Standard shipping provides better profit margins  

---

__Recommendations__
- Reduce discounts on low-margin products  
- Focus on high-profit categories like Technology  
- Improve performance in loss-making regions  
- Optimize shipping methods to reduce costs
